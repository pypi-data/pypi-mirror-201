=============
Mailing lists
=============

.. Important::

    Please report security issues **only** to
    security@hibeeproject.com.  This is a private list only open to
    long-time, highly trusted Hibee developers, and its archives are
    not public. For further details, please see :doc:`our security
    policies </internals/security>`.

Hibee has several official mailing lists on Google Groups that are open to
anyone.

.. _hibee-users-mailing-list:

``hibee-users``
================

This is the right place if you are looking to ask any question regarding the
installation, usage, or debugging of Hibee.

.. note::

    If it's the first time you send an email to this list, your email must be
    accepted first so don't worry if :ref:`your message does not appear
    <message-does-not-appear-on-hibee-users>` instantly.

* `hibee-users mailing archive`_
* `hibee-users subscription email address`_
* `hibee-users posting email`_

.. _hibee-users mailing archive: https://groups.google.com/g/hibee-users
.. _hibee-users subscription email address: mailto:hibee-users+subscribe@googlegroups.com
.. _hibee-users posting email: mailto:hibee-users@googlegroups.com

.. _hibee-developers-mailing-list:

``hibee-developers``
=====================

The discussion about the development of Hibee itself takes place here.

Before asking a question about how to contribute, read
:doc:`/internals/contributing/index`. Many frequently asked questions are
answered there.

.. note::

    Please make use of
    :ref:`hibee-users mailing list <hibee-users-mailing-list>` if you want
    to ask for tech support, doing so in this list is inappropriate.

* `hibee-developers mailing archive`_
* `hibee-developers subscription email address`_
* `hibee-developers posting email`_

.. _hibee-developers mailing archive: https://groups.google.com/g/hibee-developers
.. _hibee-developers subscription email address: mailto:hibee-developers+subscribe@googlegroups.com
.. _hibee-developers posting email: mailto:hibee-developers@googlegroups.com
