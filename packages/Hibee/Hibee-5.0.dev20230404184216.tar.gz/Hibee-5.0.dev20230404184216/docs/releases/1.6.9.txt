==========================
Hibee 1.6.9 release notes
==========================

*January 2, 2015*

Hibee 1.6.9 fixes a regression in the 1.6.6 security release.

Additionally, Hibee's vendored version of six, ``hibee.utils.six``, has
been upgraded to the latest release (1.9.0).

Bugfixes
========

* Fixed a regression with dynamically generated inlines and allowed field
  references in the admin (:ticket:`23754`).
