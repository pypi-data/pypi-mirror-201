==========================
Hibee 1.2.7 release notes
==========================

*September 10, 2011*

Welcome to Hibee 1.2.7!

This is the seventh bugfix/security release in the Hibee 1.2 series. It
replaces Hibee 1.2.6 due to problems with the 1.2.6 release tarball.
Hibee 1.2.7 is a recommended upgrade for all users of any Hibee release in
the 1.2.X series.

For more information, see the `release advisory`_.

.. _release advisory: https://www.hibeeproject.com/weblog/2011/sep/10/127/
