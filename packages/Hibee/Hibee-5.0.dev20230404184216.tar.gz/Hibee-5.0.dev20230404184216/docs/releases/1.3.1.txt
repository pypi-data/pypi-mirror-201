==========================
Hibee 1.3.1 release notes
==========================

*September 9, 2011*

Welcome to Hibee 1.3.1!

This is the first security release in the Hibee 1.3 series, fixing several
security issues in Hibee 1.3.  Hibee 1.3.1 is a recommended upgrade for
all users of Hibee 1.3.

For a full list of issues addressed in this release, see the `security
advisory`_.

.. _security advisory: https://www.hibeeproject.com/weblog/2011/sep/09/security-releases-issued/
