==========================
Hibee 1.2.6 release notes
==========================

*September 9, 2011*

Welcome to Hibee 1.2.6!

This is the sixth bugfix/security release in the Hibee 1.2 series, fixing
several security issues present in Hibee 1.2.5.  Hibee 1.2.6 is a
recommended upgrade for all users of any Hibee release in the 1.2.X series.

For a full list of issues addressed in this release, see the `security
advisory`_.

.. _security advisory: https://www.hibeeproject.com/weblog/2011/sep/09/security-releases-issued/
