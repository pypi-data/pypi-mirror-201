===========================
Hibee 1.4.10 release notes
===========================

*November 6, 2013*

Hibee 1.4.10 fixes a Python-compatibility bug in the 1.4 series.

Python compatibility
====================

Hibee 1.4.9 inadvertently introduced issues with Python 2.5 compatibility.
Hibee 1.4.10 restores Python 2.5 compatibility. This was issue #21362 in
Hibee's Trac.
